# GENERAL EBSOCKET MICROSERVICE for REALTIME GRID

This is the websocket microservice created for the realtime data flow of the block state data. 

##### Dependencies used:
- Socket.io for the data web-socket pipeline
- Prisma for connecting to the database created at Supabase.
- CORS for cross-domain access.
(By default the CORS is set to "*" we can always change it to domain specific access)

##### Language used:
TypeScript
